## タスク
## 問題

surface:117 のダッシュボードに2つの表示バグがある。

### 問題1: 全セクションの1行目が空白

Master, Conductors, Tasks, Journal すべてのセクションで、セパレーターの直後に空白行が入り、最初のコンテンツ行が見えなくなっている。

- Header行が完全に消失（1行目がいきなり Sep Master になる）
- Master セクション: ● [381] が表示されず空白
- Conductors: 3台中2台しか表示されない（1台目が空白に隠れる）
- Tasks: 最初のタスクが空白
- Journal: 最初のエントリが空白

### 問題2: Journal のフォーマット崩れ

- タイムスタンプの末尾が切れる: 03:04:19 → 03:04:1
- タイムスタンプとアイコンの間にスペースがない: 03:04:1[✓]
- タスクIDの前にスペースがない: [✓]#17747112

## 原因分析

### 問題1の原因
Sep コンポーネントが <Text> を返し、セクション本体が <Box> を返している。Ink v6 + Yoga の flex-column レイアウトで <Text> の直後に <Box> が来ると、Box の1行目が Sep と重なる。

### 問題2の原因
<Text dimColor>{entry.time} </Text> のように末尾にスペースを入れても、Ink v6 が次の <Text> との結合時にトリムしてしまう。

## 修正方針

### 問題1の修正
Sep コンポーネントの <Text> を <Box> でラップする:

```tsx
function Sep({ cols, label }: { cols: number; label: string }) {
  const line = '─'.repeat(Math.max(0, cols - label.length - 3));
  return (
    <Box>
      <Text dimColor>─ <Text bold dimColor={false}>{label}</Text> {line}</Text>
    </Box>
  );
}
```

### 問題2の修正
JournalSection の各エントリを単一の外側 <Text> に結合してスペースの消失を防ぐ:

```tsx
<Box key={i} paddingLeft={1}>
  <Text>
    <Text dimColor>{entry.time}</Text>
    {' '}
    <Text color={entry.color}>{entry.icon}</Text>
    {' '}
    <Text bold>#{entry.taskId.padStart(3, '0')}</Text>
    {' '}
    <Text>{entry.message.slice(0, maxMsg)}</Text>
  </Text>
</Box>
```

同様の問題が LogSection にもある場合は合わせて修正すること。
